Política de Privacidade
Harvest Words: Inglês Jogando

Obrigado por utilizar o Harvest Words: Inglês Jogando. Esta Política de Privacidade explica como tratamos as informações associadas ao uso deste aplicativo.

Informações Coletadas

Microfone (RECORD_AUDIO)

Utilizado exclusivamente para reconhecimento de fala. Nenhuma gravação é armazenada.

Dados de Terceiros (AdMob)

O Google pode coletar identificadores de dados de uso conforme suas próprias políticas.

Uso das Informações

  • Reconhecimento de fala
  • Exibição de anúncios
  • Nenhum dado é vendido

Segurança

O aplicativo não armazena dados pessoais em servidores próprios.

Crianças

O aplicativo não coleta intencionalmente dados de crianças.
