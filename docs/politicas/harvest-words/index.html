Política de Privacidade
Harvest Words: Inglês Jogando

Obrigado por utilizar o Harvest Words: Inglês Jogando. Esta Política de Privacidade descreve como tratamos as informações associadas ao uso deste aplicativo.

Informações Coletadas

  • Microfone (RECORD_AUDIO) – utilizado exclusivamente para reconhecimento de fala.
  • Dados de terceiros (AdMob)

Uso das Informações

  • Reconhecimento de fala
  • Exibição de anúncios
  • Nenhum dado é vendido

Segurança

O aplicativo não armazena dados pessoais em servidores próprios.

Crianças

O aplicativo não coleta intencionalmente dados de crianças.
